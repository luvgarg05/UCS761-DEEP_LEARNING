## Overview

This repository contains the practical implementations completed as part of the **UCS761 Deep Learning Laboratory**. The work focuses on understanding how neural networks function by implementing core algorithms from scratch and visualizing their behavior.

The assignments explore fundamental topics such as **activation functions, perceptron learning, neural network training using backpropagation, classification problems, and digit recognition models**.

---

## Assignments:

### 1. Activation Function Visualization

File: `102313066_Activation_Functions.py`

This program demonstrates the working of several activation functions used in neural networks. Graphs are generated to observe how each function transforms input values.

Activation functions explored include:

* Sigmoid
* Tanh
* ReLU
* Leaky ReLU
* ELU
* Softmax
* Softplus
* Binary Step

Visualization helps in understanding the differences in their output behavior.

---

### 2. Perceptron Model Implementation

File: `102313066_Perceptron_Learning.py`

This assignment focuses on implementing a **basic perceptron model** inspired by the McCulloch–Pitts neuron.

The perceptron is trained to simulate logical operations such as:

* AND
* OR
* NOR
* NOT

This experiment highlights how a single neuron can learn simple decision boundaries.

---

### 3. Neural Network Training using Backpropagation

File: `102313066_Backpropagation.py`

A **multi-layer neural network** is created and trained using the **backpropagation algorithm**.

The XOR dataset is used to illustrate how neural networks solve problems that cannot be handled by a single perceptron.

---

### 4. Binary Classification using Logistic Neuron

File: `102313066_Binary_Classification.py`

This task demonstrates binary classification using a logistic neuron trained with gradient descent.

The implementation also includes graphical visualization of:

* Decision boundaries
* Model training loss

---

### 5. Digit Recognition Model

File: `102313066_Digit_Classification.ipynb`

A neural network model is implemented for **handwritten digit classification**.

The notebook covers the full learning process including:

* Data processing
* Forward propagation
* Backpropagation training

---

### 6. Model Evaluation and Analysis

File: `102313066_Digit_Classification_Analysis.ipynb`

This notebook analyzes the performance of the digit classification model using evaluation metrics and visual tools to better understand the model’s behavior.
